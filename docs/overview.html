Lucerne is a web application framework built on Clack.

Features

Simple
Lucerne is inspired by Flask, and is designed to be small and simple: It's easy to use, and there's little to remember in order to use it.
Built on Clack
Lucerne uses Clack, an HTTP server abstraction, which enables it to use any server backend supported by Clack – from Hunchentoot the the very fast Woo.
